4.1.0.2 aifm

terminal, originally written for Adobe Illustrator 3.0+. Since Adobe Illustrator understands PostScript level 1 commands directly, you should use `set terminal post level1` instead.

Syntax: 

           set terminal aifm {color|monochrome} {"<fontname>"} {<fontsize>}
