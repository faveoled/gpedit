3.25.55 pointintervalbox

The `pointinterval` property of line types is used in plot style linespoints. A negative value of pointinterval, e.g. -N, means that point symbols are drawn only for every Nth point, and that a box (actually circle) behind each point symbol is blanked out by filling with the background color. The command pointintervalbox controls the radius of this blanked-out region. It is a multiplier for the default radius, which is equal to the point size.
