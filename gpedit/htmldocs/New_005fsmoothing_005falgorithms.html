1.4.5 New smoothing algorithms

New smoothing algorithms have been added for both 2- and 3-dimensional plots. `smooth kdensity` and `smooth cumulative` can be used with `plot` to draw smooth histograms and cumulative distribution functions. Several new smoothing kernels have been added to dgrid3d for use with `splot`. See smooth dgrid3d.
